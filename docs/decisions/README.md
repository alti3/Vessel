# Architecture Decision Records

Architecture Decision Records live in this directory.

## Naming

Use the format:

```text
ADR-0000-short-title.md
```

Numbers are monotonic and never reused. Keep titles lowercase with words separated by hyphens.

## Template

Use `ADR-0000-template.md` for new decisions.
